# Hands-On Machine Learning — Study Repository

This repository documents my study of:

**Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow (3rd Edition)**  
by Aurélien Géron

---

##  Objective

- Learn Machine Learning through hands-on practice  
- Build a strong understanding of core ML concepts  
- Write clean, readable, and well-structured code  
- Develop a solid foundation for real-world projects  

---

##  Learning Approach

This repository follows the structure of the book and is organized **by chapter** for clarity.

At the same time, the content can be grouped into two main areas:

- **Fundamentals of Machine Learning**  
  (data preprocessing, classical models, evaluation)

- **Neural Networks & Deep Learning**  
  (neural architectures, training, TensorFlow/Keras)

Each chapter includes:
- practical implementations  
- experiments and visualizations  
- concise explanations of key concepts
